# Crypto Portfolio Tracker

A professional crypto tracking application that provides real-time updates on asset performance and total portfolio value.

## Features
- **Portfolio Overview:** Track your total net worth and profit/loss in real-time.
- **Performance Charts:** Visualize portfolio growth with interactive area charts.
- **Asset Allocation:** Understand your portfolio distribution with pie charts.
- **Market Insights:** Real-time feedback on market trends.

## Setup

### App (Next.js)
1. Go to `root`
2. Run `npm install`
3. Run `npm run dev`

Note: This project is a frontend-only demonstration with mock data integration in `lib/crypto-api.ts`.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Lucide React, Recharts
- **Data Integration:** Mock API integration for high-performance UI demonstration.
