# PRD: Crypto Portfolio Tracker

## 1. Project Overview
A crypto tracking application that connects to various exchange APIs (Binance, Coinbase, Kraken) to provide real-time updates on asset performance and total portfolio value.

## 2. Target Audience
- Individual Crypto Traders
- Investment Groups
- Financial Analysts

## 3. Key Features
- **Real-Time Price Updates:** Accurate asset prices and portfolio value tracking.
- **Profit/Loss Charts:** Visualize gains and losses over different timeframes.
- **Wallet Integration:** Support for major exchange APIs and cold wallets.
- **Tax Reporting:** Generate simple tax reports based on transaction history.

## 4. Tech Stack (CV Reference)
- **Frontend:** Next.js and Tailwind CSS (Highly performant, modern UI).
- **Backend:** Node.js with exchange API SDKs.
- **Data Viz:** Chart.js or D3.js for interactive charts.
- **Storage:** PostgreSQL for transaction and user data.

## 5. Roadmap
- **MVP:** Manual asset entry and basic price tracking.
- **V1:** Exchange API integrations and portfolio charts.
- **V2:** DeFi wallet support and AI-driven market insights.
